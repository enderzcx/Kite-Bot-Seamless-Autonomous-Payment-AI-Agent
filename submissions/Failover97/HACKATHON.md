# 逆转人格

GitHub User: Failover97
ProjectName: 逆转人格
Track: LLM赛道
ProjectDescription: **逆转人格**是一个基于大语言模型的人格驱动决策对抗系统，旨在解决 LLM 在复杂决策场景中过度迎合用户、放大认知偏差的问题。项目将 MBTI 人格结构映射为可量化的决策偏好模型（风险承受、交易频率、决策驱动因子等），并自动生成与用户人格在核心维度上完全对立的“镜像人格 Agent”。用户与镜像人格围绕投资决策在多个关键维度（入场逻辑、时机、仓位、风险预案、退出策略）进行多轮对抗，由裁判 Agent 根据论据完整性与决策可执行性动态评分，并以数值化反馈暴露用户决策盲点。
该系统的目标不是给出“对或错”的答案，而是通过结构化对抗与量化反馈，提升用户的决策质量与风险认知能力。
